# 宠物消消乐

## 简介
宠物消消乐是一款经典的三消类网页游戏，玩家通过交换相邻的宠物图标，使三个或更多相同的宠物连成一线即可消除并得分。游戏包含多个关卡，每个关卡有不同的BOSS和挑战目标。

## 技术栈
- HTML5: 构建网页结构
- CSS3: 设计游戏界面和动画效果
- JavaScript (ES6+): 实现游戏逻辑和交互功能

## 主要功能
- 动态加载关卡资源（图片、音效）
- 匹配检测与消除逻辑
- 单人闯关模式
- BOSS特效与音效
- 分数统计与关卡切换

## 关键算法
- 匹配检测：水平和垂直方向检查连续三个相同元素的匹配情况
- 消除与填充：标记匹配项并触发消除动画，将上方元素下落填补空位

## 扩展功能
- 支持多人对战模式
- 添加排行榜和成就系统

## 运行环境
- 浏览器支持：Chrome, Firefox, Safari, Edge
